# plugin-tool-math — 数学计算工具集

**包名**: `@aalis/plugin-tool-math`  
**源码**: `packages/plugin-tool-math/src/index.ts`

## 概述

覆盖 10 个数学类别的计算工具集，AI 可进行精确数值计算而非依赖 LLM 估算。

## 插件声明

```typescript
meta.name = '@aalis/plugin-tool-math'
meta.inject = {}
```

## 计算类别

| 类别 | 说明 |
|---|---|
| 表达式求值 | 数学表达式解析与计算 |
| 统计分析 | 均值/中位数/方差/标准差等 |
| 矩阵运算 | 矩阵乘法/转置/行列式/逆矩阵 |
| 数论 | 质数判定/因数分解/GCD/LCM |
| 几何计算 | 面积/体积/距离/角度 |
| 单位转换 | 长度/质量/温度/时间等单位互转 |
| 金融计算 | 复利/年金/NPV/IRR |
| 微积分 | 数值积分/微分 |
| 方程求解 | 一元/多元方程求解 |
| 进制转换 | 二/八/十/十六进制互转 |
