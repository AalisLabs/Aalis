示例前端 · webui-client-example

🧩 你正在看「第二前端」

这是一个最小的演示前端。能看到它，说明 Aalis 的「忒修斯之船——前端可热切换」整条链路是通的：

  1. 它的 package.json 标了 aalis.client: true，被 webui-server 动态发现（没有任何硬编码包名）；
  2. 你在默认前端的 Dashboard「前端（活跃切换）」卡片里把它设为活跃；
  3. webui-server 实时重挂静态目录并持久化 webui.client，页面 reload 后由它接管。

切回默认前端：默认前端在 @aalis/plugin-webui-client。回到它的 Dashboard 同一张卡片选回即可 （若当前这个示例前端没有切换卡片，直接改配置 webui.client 或访问默认前端 URL）。
